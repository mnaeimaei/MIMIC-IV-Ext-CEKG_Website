General Overview of the Dataset

In this dataset, each event is related to a timestamp, an activity, a patient identifier, and its admission or outpatient ID. There are also connections between the admission ID and diagnoses of disorders based on ICD codes. Activities have different features, and features have different values.

  • Number of events: 121,577,735
  • Number of distinct patients: 282,484
  • Number of distinct admission IDs: 448,709
  • Number of distinct outpatient IDs: 550,405
  • Number of distinct disorders based on high-level ICD: 1,759
  • Number of distinct activities: 95
