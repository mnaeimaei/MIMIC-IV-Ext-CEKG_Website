Accessing the Dataset Files

To access the dataset files associated with MIMIC-IV-Ext-CEKG, please follow the steps below:

  1. Create an account on PhysioNet.
  2. Apply to become a credentialed user via this application page.
  3. Complete the required training on human research and data privacy by following the instructions on the CITI Course page.
  4. Sign the Data Use Agreement (DUA) specific to this dataset: Sign DUA.
